### Flow 4 Detail (Broken)

`sidebar:task-retry` IPC handler calls `agentManager.retryTask()` which auto-resubmits silently. The `overlay:prefill-instruction` IPC channel is wired on the receiver side (preload + renderer) but the sender in `ipc-handlers.ts` never sends it. Retry works functionally (task re-runs) but the overlay textarea prefill path (D-19) is disconnected.

**Severity:** Non-critical. Retry functionality works; only the "prefill then let user edit before re-submitting" UX path is missing.
